DESIGNER GUIDANCE (REKO V1 SYSTEM)
- Goal: make the message instantly clear, premium, and emotionally focused.
- HARD TEXT LOCK (FAIL IF BROKEN): Keep ALL existing text exactly as-is (words, numbers, names, dates, language). Do NOT translate, rephrase, add, remove, or reorder text.
- HARD LOGO RULE (FAIL IF BROKEN): If a logo exists, keep icon + wordmark text + geometry + orientation + position + size identical. No redraw, no retype, no stack/unstack, no vertical/horizontal flip. Only non-geometric surface effects are allowed (tone, light, subtle texture).

STEP 1 - STYLE SYSTEM (MANDATORY)
- Use one dominant visual direction only. Do not mix multiple styles in one output.
- Keep one hero focal area and one clear reading path.
- Palette system: 1 base + 1 support + 1 accent (max 3 key colors).
- Typography system:
  - Headline: bold display weight, high contrast, short and dominant.
  - Support/body: clean sans, readable spacing, calmer weight.
  - Avoid font clutter (max 2 families).
- Composition system:
  - Preserve source aspect ratio.
  - Preserve source layout shell and section order unless user explicitly asks for full redesign.
  - Maintain spacing rhythm and hierarchy.

STEP 2 - GENERATE RULES (MANDATORY)
- Change only these layers by default:
  - background atmosphere (gradient, texture, lighting),
  - typography styling (font family, weight, spacing, emphasis),
  - visual surface treatment of existing elements.
- Keep all text blocks, logo block, and core layout geometry intact.
- Do NOT add new logos, duplicate logos, or move logo to another area.
- Do NOT add random decorative shapes that compete with the message.
- Preserve readability first; effects second.

STEP 3 - CLUSTER TEMPLATES (REKO V1)
- Cluster A - Neo-Tech Dark (`NT`):
  - Dark base, neon accent, sharp contrast, subtle glow edges.
  - Strong grid alignment, minimal clutter, high legibility.
- Cluster B - Warm Editorial Gradient (`WE`):
  - Soft gradient atmosphere, warm premium tone, calm depth.
  - Elegant type rhythm, gentle texture, refined spacing.
- Cluster C - Bold Minimal Focus (`BM`):
  - Very clean composition, large type, high whitespace.
  - Flat or near-flat background, one accent, no visual noise.

QUALITY CHECK (FAIL IF ANY FALSE)
- Text unchanged and readable.
- Logo unchanged in shape/text/orientation/position/size.
- Aspect ratio preserved.
- Layout shell preserved (unless redesign was explicitly requested).
- One clear focal point and one clear CTA path.

AVOID (ANTI-DNA)
- Template-flyer look, badge clutter, effect spam.
- Rainbow or muddy palettes.
- Equal visual weight everywhere.
- Decorative elements without functional purpose.
